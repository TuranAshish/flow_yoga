# 🧘 Flow Yoga

A modern, responsive yoga and meditation website designed to help users find balance, improve wellness, and build healthy habits through guided yoga practices.

🌐 Live Demo: https://flowing-yoga.vercel.app/

---

## ✨ Features

- Beautiful modern UI/UX
- Fully responsive design (Mobile, Tablet, Desktop)
- Hero section with call-to-action
- Yoga classes showcase
- Benefits and wellness sections
- Testimonials from practitioners
- Contact form
- Smooth scrolling navigation
- Fast loading performance
- SEO-friendly structure
